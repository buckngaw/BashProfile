Alias

alias เป็นคำสั่งที่ทำให้เราจำลองคำสั่งของเครื่องเดิมเป็นคำสั่งใหม่ที่เราตั้งเอง (อะไรก็ได้แต่ต้องไม่ไปซ้ำกับของเดิมนะ) ของระบบ Unix,Linux


วิธีใช้ alias

คำส่ั่งใหม่ที่เราตั้งเอง = "คำสั่งที่ต้องการให้ทำงานจริงๆ" เช่น

alias copy="cp" เป็นการสั่งให้ copy เป็นการก๊อบปี้ไฟล์เหมือนคำสั่ง cp

alias ls="ls -GF" เป็นการสั่งให้ ls แสดงสีและเครื่องหมายบอกชนิดไฟล์เหมือนคำสั่ง ls -GF


วิธียกเลิก alias ที่ตั้งใหม่ ทำได้โดยใช้

unalias ชื่อคำสั่งที่ตั้งใหม่

เช่น unalias copy เป็นการยกเลิกคำสั่ง copy กลับไปใช้ cp เหมือนเดิม
